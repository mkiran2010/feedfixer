# Extension icons

Drop the FeedFixer logo here at exactly these filenames + sizes:

- `icon-16.png` — 16×16 (toolbar, favicon)
- `icon-32.png` — 32×32 (Windows)
- `icon-48.png` — 48×48 (extensions page)
- `icon-128.png` — 128×128 (Chrome Web Store, install dialog)

PNG with transparent background recommended.

When all four exist, uncomment the `icons` block in `manifest.json` at the project root.
